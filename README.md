# AI Joke Generator 

A simple Python CLI application that generates jokes using the Google Gemini AI API.  
The program allows users to enter a topic and receive a funny joke generated by AI.  
All generated jokes are also saved to a text file for later reference.

---

## Project Overview

This project demonstrates how to integrate a Generative AI API into a Python application.  
The program interacts with the Gemini AI model to dynamically generate jokes based on user input.

It also demonstrates important programming concepts such as:

- API integration
- Python loops
- user input handling
- file handling
- functions
- basic CLI application development

---

## Features

- Generate AI-powered jokes based on user input
- Continuous joke generation using a loop
- Save all generated jokes to a file
- Clean terminal output
- Beginner-friendly Python project

## Installation

### 1. Clone the repository

```

git clone [https://github.com/yourusername/ai-joke-generator.git](https://github.com/yourusername/ai-joke-generator.git)

```

Navigate into the project folder:

```

cd ai-joke-generator

```

---

### 2. Install dependencies

Install the Gemini Python library:

```

pip install google-genai

````

---

### 3. Get a Gemini API Key

1. Go to Google AI Studio  
https://aistudio.google.com/app/apikey  

2. Create an API key

3. Copy the key

---

### 4. Add the API key

Replace the placeholder in the code:

```python
client = genai.Client(api_key="YOUR_GEMINI_API_KEY")
````

with your API key.

---

## Running the Program

Run the program using:

```

python3 main.py

```

Example interaction:

```

Enter a topic (or type quit): cats

Joke:
Why don't cats play poker in the jungle?
Too many cheetahs!

------------------------------

```

Type `quit` to exit the program.

---

## How It Works

1. The program asks the user for a topic.
2. A prompt is generated using the topic.
3. The prompt is sent to the Gemini AI model.
4. The AI generates a joke.
5. The joke is printed in the terminal.
6. The joke is saved to a file.
7. The program repeats until the user exits.
